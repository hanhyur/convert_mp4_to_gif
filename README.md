# convert_mp4_to_gif
mp4 gif 변환기

## 실행방법 How to run the program

convert.py가 저장된 위치에서 command 창으로 아래의 커맨드를 입력

In the directory where `convert.py` is saved, enter the following command in the cmd.

```
python .\mp4_to_gif_gui_ffmpeg.py
```
